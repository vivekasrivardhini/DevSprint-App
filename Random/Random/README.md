
# 🚀 Team Random - Galactic Command Center

A single-screen mobile app built in **2.5 hours** for the Mobile App Challenge. 
This app synthesizes a unique intergalactic identity by combining 4 distinct APIs.

Video link: https://drive.google.com/file/d/1Vzmfb7lrBxudf1nst6NecSQkACn1Tb6O/view?usp=drive_link

## 🔌 APIs Integrated
1. **Creative (Robohash):** Generates a unique alien/robot avatar based on a random seed.
2. **Utility (Agify):** Predicts "Chrono-Age" based on character name.
3. **Humor (Evil Insult):** Fetches a "Battle Cry" for your galactic warrior.
4. **Science (Spaceflight News):** Real-time mission briefings pulled from actual space news.

## 🛠️ Tech Stack
- **Language:** Swift 5.10 / SwiftUI
- **Architecture:** MVVM with Service Layer
- **Concurrency:** Swift `async/await` and `TaskGroup` for parallel API fetching.
